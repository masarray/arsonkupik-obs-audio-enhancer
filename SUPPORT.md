## For users

Start here:

- Download and install: `docs/USER_INSTALL.md`
- Windows build: `docs/BUILD_WINDOWS.md`
- Build troubleshooting: `docs/BUILD_TROUBLESHOOTING.md`
- Release workflow: `docs/RELEASE_WORKFLOW.md`

## Common user flow

1. Download the latest release ZIP.
2. Extract it.
3. Close OBS Studio.
4. Run `install_windows.bat`.
5. Restart OBS Studio.
6. Add the filter from `Audio Source > Filters > + > ArSonKuPik Smart Enhancer`.

## Issue types

Use GitHub Issues for:

- Install problems
- OBS compatibility problems
- Audio artifacts or clipping
- Preset tuning suggestions
- Workflow/build failures
- Documentation fixes

For security issues, read `SECURITY.md` first.
