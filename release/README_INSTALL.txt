ArSonKuPik OBS Audio Enhancer - Windows x64
================================================

How to install
--------------
1. Close OBS Studio.
2. Extract this ZIP file.
3. Double-click install_windows.bat.
4. Allow Administrator permission if Windows asks.
5. Start/restart OBS Studio.
6. Add the filter from:
   Audio Source > Filters > + > ArSonKuPik Smart Enhancer

How to uninstall
----------------
1. Close OBS Studio.
2. Double-click uninstall_windows.bat.
3. Restart OBS Studio.

Manual install path
-------------------
Copy the folder:
  arsonkupik-obs-audio-enhancer

into:
  C:\ProgramData\obs-studio\plugins\

Expected final layout:
  C:\ProgramData\obs-studio\plugins\arsonkupik-obs-audio-enhancer\bin\64bit\arsonkupik-obs-audio-enhancer.dll
  C:\ProgramData\obs-studio\plugins\arsonkupik-obs-audio-enhancer\data\locale\en-US.ini

Recommended default setting
---------------------------
Preset: MasAri
Manual macro tuning: OFF
Smart Protect: ON
Bypass: OFF

Notes
-----
This is a native OBS audio filter package for Windows x64.
If OBS was open during installation, restart OBS before looking for the filter.
