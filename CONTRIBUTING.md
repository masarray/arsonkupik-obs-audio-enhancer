# Contributing

Thanks for your interest in contributing to ArSonKuPik OBS Audio Enhancer.

## How to contribute
- Open an issue before large feature work.
- Keep pull requests focused and easy to review.
- Include test notes, platform notes, and screenshots when relevant.
- For DSP changes, describe the listening goal and expected audible effect.

## Development expectations
- Prefer practical, low-CPU DSP solutions suitable for OBS live usage.
- Avoid changes that make presets sound louder at the cost of obvious distortion.
- Preserve user-facing simplicity.

## Pull request checklist
- [ ] Builds locally
- [ ] No bundled binaries or build artifacts committed
- [ ] Docs updated when behavior changes
- [ ] Release notes updated for user-visible changes
