Native OBS Studio audio filter

Smart audio enhancement that sounds better—not merely louder.

ArSonKuPik gives music, media, podcasts, and spoken audio a more polished, alive, and engaging presentation while protecting real-time stability. Its macro controls are deliberately audible, but gain staging is calibrated to avoid the huge hidden level jump common in “wow effect” processing.

Engineering priorities

Built around stable listening quality

ArSonKuPik treats real-time audio behavior as a product requirement. Parameter changes preserve filter state, continuous controls are smoothed, preset topology changes are masked by a short transition, and dynamic gain behavior is intentionally constrained.

FAQ

Common questions

Is ArSonKuPik free?

Yes. The project is free and open source under the GNU General Public License v3.0.

How loud should Filter ON be?

The current gain-staging target is a tasteful audible benefit rather than an extreme jump. The main project smoke test targets roughly 3–4 dB of peak benefit.

Will moving a slider create crackle?

The current engine preserves filter history and smooths continuous parameters to prevent the state resets and zipper noise that caused earlier crackle.

Does it require powerful hardware?

The engine is designed for practical live use and avoids making global oversampling the default. Actual CPU use still depends on the system, sample rate, sources, and OBS scene complexity.

Does sponsoring unlock extra features?

No. Donations are voluntary project support. The public GPL-3.0 plugin remains complete, and sponsorship does not buy technical control, guaranteed deadlines, or private features.
