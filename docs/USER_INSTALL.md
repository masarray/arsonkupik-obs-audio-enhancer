# User install guide

This document is for end users who only want to install the compiled OBS plugin.

## Download

Open the latest GitHub Release and download:

```text
ArSonKuPik-OBS-Audio-Enhancer-Windows-x64.zip
```

## Install

1. Close OBS Studio.
2. Extract the ZIP file.
3. Double-click `install_windows.bat`.
4. Approve Administrator permission.
5. Start OBS Studio.
6. Add the filter from:

```text
Audio Source > Filters > + > ArSonKuPik Smart Enhancer
```

## Default setting

```text
Preset: MasAri
Manual macro tuning: OFF
Smart Protect: ON
Bypass: OFF
```

## Uninstall

Close OBS Studio and double-click:

```text
uninstall_windows.bat
```

## Manual install

Copy the folder:

```text
arsonkupik-obs-audio-enhancer
```

into:

```text
C:\ProgramData\obs-studio\plugins\
```

Final layout:

```text
C:\ProgramData\obs-studio\plugins\arsonkupik-obs-audio-enhancer\bin\64bit\arsonkupik-obs-audio-enhancer.dll
C:\ProgramData\obs-studio\plugins\arsonkupik-obs-audio-enhancer\data\locale\en-US.ini
```
