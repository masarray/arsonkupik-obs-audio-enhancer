# ArSonKuPik OBS Audio Enhancer v0.4.12

## Highlights
- Recalibrates gain staging after the stronger v0.4.11 VST-style macro controls.
- Keeps the audible knob movement, but prevents Filter ON from becoming a large hidden volume boost.
- Adds an internal -8 dB creative-chain headroom trim before the DSP rack, with only partial calibrated restore afterward.

## What changed
- The creative rack now receives lower internal level so EQ, color, compressor, and width stages have cleaner headroom.
- Smart makeup is capped lower and rises more conservatively.
- Compressor makeup and parallel density are reduced so bass-heavy content does not feel like it is pumping.
- Output gain mapping is recalibrated so the main MasAri preset targets roughly +3 to +4 dB benefit in the smoke test instead of a very large level jump.
- The DSP smoke test now fails if the main preset moves outside the calibrated target or if any preset creates an extreme hidden boost.

## Test result
- MasAri smoke-test peak benefit: about +3.41 dB.
- SonKuHoreg smoke-test peak benefit: about +3.48 dB.
- SonKuBalap smoke-test peak benefit: about +3.92 dB.
- SonKuBattle remains the hottest stress preset at about +4.63 dB, still below the extreme boost guard.
- Clipping: no.

## Notes
This release intentionally does not weaken the v0.4.11 macro feel. The fix is gain staging: more internal headroom, lower restore, and tighter smart makeup.
