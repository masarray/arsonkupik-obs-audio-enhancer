# ArSonKuPik OBS Audio Enhancer v0.4.11

## Highlights
- Ported the stronger askp-vst style macro control behavior into the OBS engine.
- Smart Bass, Smart Treble, Vocal Body, and Stereo Magic now react more clearly when moved from minimum to maximum.
- Kept the v0.4.10 crackle-free smoothing, CPU-safe dirty apply behavior, and stable no-pumping makeup guard.

## What changed
- Smart Bass is now bipolar: left cleans rumble/mud, center is neutral, right adds sub/body/punch.
- Smart Treble is now bipolar: left smooths harshness/sibilance, center is neutral, right adds air/presence/sparkle.
- Vocal Body is now bipolar: left tucks/smooths vocal, center is neutral, right moves vocal forward with body and clarity.
- Stereo Magic is now bipolar: left narrows toward mono, center is raw/neutral, right widens/open side air.
- Macro EQ lanes stay allocated in a stable order so slider automation remains crackle-safe.

## Testing focus
- Sweep each knob from 0 to 100 while audio is playing.
- Confirm the effect is clearly audible without zipper noise.
- Confirm bass-heavy sources do not create obvious pumping.
- Confirm Stereo Magic at minimum narrows strongly and at maximum opens the image.

## Notes
This release is a feel/behavior release. It makes the four main creative knobs more useful while preserving the crackle-free and CPU-conscious architecture from v0.4.10.
