# ArSonKuPik OBS Audio Enhancer v0.4.6

## Highlights
- Fixed the release workflow status by removing GitHub Pages deployment from tag-based release runs.
- Release workflow now focuses only on release deliverables: Windows installer, Windows portable ZIP, Linux archive, and GitHub Release notes.
- GitHub Pages deployment remains separated into the Pages workflow so release tags are not blocked by environment protection rules.

## Recommended downloads

### Windows
- Most users: download the installer `.exe`.
- Advanced users: download the portable `.zip` and copy the plugin folder manually.

### Linux
- Download the `.tar.gz` package and install manually according to your OBS setup.

## Notes
This release is a release-automation hotfix. Audio behavior is unchanged from v0.4.5.
