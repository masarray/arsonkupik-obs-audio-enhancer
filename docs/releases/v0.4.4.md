# ArSonKuPik OBS Audio Enhancer v0.4.4

## Highlights
- Cleaned repository helper scripts for a more professional public repo.
- Confirmed GPL-3.0 licensing.
- Improved Windows installer behavior for standard OBS plugin installation.
- Installer now installs into the standard ProgramData OBS plugin directory without warning just because the folder already exists.
- Release automation continues to publish Windows installer, Windows portable ZIP, and Linux archive.

## Recommended downloads

### Windows
- Most users: download the installer `.exe`.
- Advanced users: download the portable `.zip` and copy the plugin folder manually.

### Linux
- Download the `.tar.gz` package and install manually according to your OBS setup.

## Notes
Close OBS before installing or updating the plugin. Restart OBS after installation.
