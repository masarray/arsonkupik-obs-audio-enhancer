# ArSonKuPik OBS Audio Enhancer v0.4.5

## Highlights
- Fixed Linux release build failure caused by linking a non-PIC static DSP library into the OBS shared module.
- `arsonkupik_dsp` is now built with position-independent code for Linux `.so` packaging.
- Linux release configure step now sets `CMAKE_BUILD_TYPE=Release`.
- CI now includes a Linux OBS plugin build smoke test so this class of release error is caught before publishing.

## Recommended downloads

### Windows
- Most users: download the installer `.exe`.
- Advanced users: download the portable `.zip` and copy the plugin folder manually.

### Linux
- Download the `.tar.gz` package and install manually according to your OBS setup.

## Notes
This release is a packaging/build-system hotfix. Audio behavior is unchanged from the previous public-ready build.
