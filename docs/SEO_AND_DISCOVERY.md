```text

```

## Release description format

```markdown
Native OBS Studio audio enhancer plugin for streaming, music, podcast, karaoke, and YouTube creators.

Highlights:
- MasAri reference preset
- Smart Bass / Deep Horeg
- Smart Treble / Airy Krenyes
- Vocal Body
- Stereo Magic
- Smart Protect limiter
- One-click Windows installer

Install:
1. Download the Windows x64 ZIP.
2. Extract it.
3. Run install_windows.bat.
4. Restart OBS.
5. Add the filter from Audio Source > Filters > + > ArSonKuPik Smart Enhancer.
```

## README SEO structure

The README should keep these sections near the top:

1. Product name with `OBS Audio Enhancer` in the H1.
2. One-sentence description with `OBS Studio`, `audio filter`, `streaming`, and `presets`.
3. Download/install section before developer build details.
4. Feature table with user-facing use cases.
5. Keywords section near the bottom.
6. License and support sections.

## Trust signals

Keep these visible:

- Build badges
- Release badge
- Apache-2.0 license badge
- Installer instructions
- Security policy
- Contributing guide
- Changelog
- Clear OBS independence disclaimer

## What not to do

- Do not overpromise professional mastering replacement.
- Do not claim official OBS endorsement.
- Do not hide the installation path.
- Do not make the README only developer-focused.
- Do not use too many technical DSP terms before explaining user benefits.
