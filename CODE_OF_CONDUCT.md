# Code of Conduct

## Our pledge

We want ArSonKuPik OBS Audio Enhancer to be a friendly and productive open-source project for creators, audio engineers, streamers, developers, and beginners.

We pledge to make participation welcoming, respectful, and harassment-free.

## Expected behavior

- Be respectful and constructive.
- Focus feedback on the code, documentation, audio result, or user experience.
- Explain audio observations clearly, such as bass, harshness, clipping, phase, stereo width, vocal clarity, or long-listening fatigue.
- Help beginners when possible.
- Assume good intent, but accept corrections gracefully.

## Unacceptable behavior

- Harassment, insults, threats, or personal attacks.
- Discriminatory language or conduct.
- Publishing private information without consent.
- Repeated off-topic arguments.
- Spam, malware, or intentionally misleading contributions.

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, or block participants who violate this code of conduct.

## Attribution

This code of conduct is inspired by common open-source community standards and adapted for this project.
